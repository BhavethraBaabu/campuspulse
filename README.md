# CampusPulse Clark

> Turned Clark University's confusing website into a conversational AI assistant.

https://campuspulse-bice.vercel.app
https://github.com/BhavethraBaabu/campuspulse


---

## What It Does

Clark students spend 30–45 minutes searching a fragmented website for basic information. CampusPulse fixes that — one conversational interface, any Clark question, cited answer in under 2 seconds.

**285 pages indexed · 1.8s response · 6 languages · $0/month**

---

## Stack

`Next.js 14` `TypeScript` `Tailwind CSS` `Framer Motion` `Groq Llama 3.1` `Qdrant` `Supabase` `Vercel`

---

## Features

- **AI Chat** — RAG over 285 Clark pages, cited answers in < 2s
- **Voice Input** — Web Speech API
- **Course Recommender** — AI-matched courses based on interests + goals
- **Live Deadlines** — real-time countdown timers
- **GPA Calculator** — Clark's official 4.0 scale
- **Email Templates** — OPT, grade appeal, financial aid, late withdrawal
- **Study Planner** — graduation, finals, OPT timelines
- **Campus Map + Directory** — every building, office, staff member
- **Multi-language** — English, Spanish, Hindi, Chinese, Arabic, French
